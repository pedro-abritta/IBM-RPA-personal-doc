# Comando: Dividir texto

## Descrição
Divide todos os valores separados por um delimitador especificado (normalmente é vírgula)

## Print do Fluxo![[dividir_texto.png]]
*Legenda: tool 'Dividir texto'

## Sintaxe/Parâmetros
- **Texto para divisão**: variável que será dividida
- **Tipo de delimitador**: o que definirá a separação dos valores
	- **delimitador customizado**: vírgula, ponto, ponto e vírgula....
- **Limite de subtextos**: quantidade de valores máximo
- **Lista de textos**: formato final é uma lista com um valor em cada índice
