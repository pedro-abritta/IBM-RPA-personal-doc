# Comando: Enviar para o sistema de arquivos

## Descrição
envia arquivos para OneDrive

## Print do Fluxo![[enviar_sistem_arquivos.png]]
*Legenda: tool 'Enviar para o sistema de arquivos'

## Sintaxe/Parâmetros
- **Sistema de arquivos**: conexão criada com onedrive
- **origem**: diretório local do arquivo que será enviado
- **destino**: qual a pasta no OneDrive que será salvo o arquivo
## Links Relacionados
- [[conectar_microsoft_onedrive]] usado para criar uma conexão no OneDrive

## Observações Práticas
- tudo relacionado à tools de diretórios locais é possível utilizar no OneDrive, para isso é necessário passar a conexão criada (onedriveCoon) por parâmetro dentro de 'sistema de arquivos'
