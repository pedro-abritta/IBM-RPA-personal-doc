# Comando: Conectar ao Microsoft OneDrive

## Descrição
fazer conexão com uma conta OneDrive

## Print do Fluxo![[conectar_onedrive.png]]
*Legenda: tool 'Conectar ao microsoft OneDrive'

## Sintaxe/Parâmetros
- **Nome**: foi criado no tenant em 'connections'
- **Conexão**: output que cria a conexão do OneDrive

## Links Relacionados
- [[enviar_sistema_arquivos]] após fazer a conexão, usa-se para submeter e buscar arquivos na nuvem

## Observações Práticas
- tudo relacionado à tools de diretórios locais é possível utilizar no OneDrive, para isso é necessário passar a conexão criada (onedriveCoon) por parâmetro dentro de 'sistema de arquivos'
